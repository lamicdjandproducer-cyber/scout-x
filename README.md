# Scout X 🏆

AI-powered sports betting analyst on WhatsApp.

## Stack
- Node.js + TypeScript
- OpenAI GPT-4o (tool calling, Whisper, Vision)
- API-Sports (football, basketball, MMA)
- SportAPI (live events, transfers, rankings)
- The Odds API (multi-bookmaker odds)
- Z-API (WhatsApp)
- Stripe (subscriptions)
- PostgreSQL on Railway

## APIs
- Football, H2H, standings, injuries, player stats
- NBA games and standings
- UFC/MMA events and fighters
- Live events across all sports
- Real-time odds + value bet detection
- Transfer news
- Audio transcription (Whisper)
- Betting slip analysis (GPT-4o Vision)
